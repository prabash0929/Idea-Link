#🎓 Idea Link – Project Approval System (PAS)

## 📌 Project Description

**Idea Link (PAS)** is a secure, web-based Project Approval System developed using ASP.NET Core.
The system enables students to submit project proposals and allows supervisors to review them through a **blind matching mechanism**, ensuring fair and unbiased selection.

---

## 🚀 Technologies Used

* ASP.NET Core Web API
* C#
* HTML
* CSS
* Entity Framework Core
* SQL Server

---

## ✨ Key Features

* 🔐 Secure Login System (Role-Based Access)
* 👨‍🎓 Student Project Submission & Management
* 👨‍🏫 Supervisor Blind Review Dashboard
* 🔄 Blind Matching Mechanism
* 👁️ Identity Reveal after Match Confirmation
* 🛠️ Admin & System Administration Control Panel
